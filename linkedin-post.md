# LinkedIn Post

I have started a new beginner cloud security portfolio project: Azure Cloud Security Review Lab.

The aim of this project is to connect my SOC and Blue Team learning with cloud security fundamentals.

The lab focuses on:

- Microsoft Entra ID
- MFA
- Conditional Access
- Privileged access
- Risky sign-ins
- Identity security
- Microsoft Sentinel logging concepts
- Account compromise response
- Cloud security recommendations

This project helped me practise how a junior cloud security or cyber security analyst might review a small cloud environment, identify common risks, document findings, and recommend practical improvements.

My long-term goal is to continue developing in Cyber Defence, Managed Security Services, SOC-style work, and Microsoft security technologies. I believe cloud security knowledge is an important foundation for modern security operations and incident response.
